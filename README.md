# Book-Recommendation-System
A content-based book recommendation system using TF-IDF and cosine similarity to suggest similar books.

This project is a content-based book recommender system.
It recommends books similar to a given title by analyzing title, author, and tags using TF-IDF Vectorization and Cosine Similarity.

Features:

Content-based filtering (no user history required)

Uses TF-IDF Vectorization to represent book features

Cosine Similarity is used to measure how close two books are based on their TF-IDF vector representations.

Recommends top-N similar books to a given title

Built with pandas and scikit-learn
